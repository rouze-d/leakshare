LEAKSAHRE : SIMPLE FIND LEAK INFORMATION

Leakshare is simple tool for find leak information from DarkWeb.

need:
```
sudo gem install lolcat
sudo apt install tor curl
```

usage:
```
bash leakshare.sh
```
and put domain target
